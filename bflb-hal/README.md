# HAL for BouffaloLab chips

## BL808
The Bouffalo Lab BL808 contains 3 RISC-V CPUs:
- M0: 32-bit RISC-V RV32IMAFCP
- D0: 64-bit RISC-V RV64IMAFCV (C906)
- LP: 32-bit RISC-V RV32EMC
